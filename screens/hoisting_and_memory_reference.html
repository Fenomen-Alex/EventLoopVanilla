Heap vs. Stack

view_agenda Call Stack

Stores primitives (String, Number, Boolean) and function execution contexts. Fixed memory size, LIFO structure.

scatter_plot Memory Heap

Stores reference types (Objects, Arrays, Functions). Dynamically allocated, unstructured memory pool. Stack holds pointers to Heap addresses.

warning

Temporal Dead Zone (TDZ)

The TDZ is the period from the start of a block until the initialization of a let, const, or class variable. Accessing the variable before initialization throws a ReferenceError.

// Global Execution Context const globalVar = "I am safe"; { // Block Scope Starts -> TDZ for 'data' Begins console.log(globalVar); // OK console.log(data); // ReferenceError! Inside TDZ let data = "Initialized"; // TDZ Ends Here console.log(data); // OK: "Initialized" }
